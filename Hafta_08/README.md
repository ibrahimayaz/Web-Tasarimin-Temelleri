# HAFTA 8: CSS Metin ve Font Özellikleri

## Teorik İçerik
- Font özellikleri (font-family, font-size, font-weight)
- Font-style ve font-variant
- Text özellikleri (text-decoration, text-align)
- Text-transform, text-indent
- Line-height ve letter-spacing
- Color ve background-color
- Ölçü birimleri (px, em, rem, %, vw, vh)

## Pratik Uygulamalar
- Farklı font kombinasyonları
- Metin biçimlendirme örnekleri
- Tipografi çalışmaları
- Responsive font boyutları

## Ödevler
- Tipografi odaklı bir blog sayfası tasarlayın
- En az 3 farklı font ailesi kullanın
- Okunabilirliği optimize edin

## Değerlendirme
- Tipografi projesi incelemesi
- Tasarım kalitesi değerlendirmesi
