# HAFTA 5: HTML Form Elemanları ve Validasyon

## Teorik İçerik
- Form input tipleri: number, color, date, file
- Radio button ve checkbox kullanımı
- Dropdown menü (<select>, <option>)
- Textarea kullanımı
- Form butonları (submit, reset)
- HTML5 form validasyonu
- Required, pattern, maxlength özellikleri
- GET ve POST metotları

## Pratik Uygulamalar
- Kapsamlı kayıt formu oluşturma
- Form validasyon örnekleri
- Dosya yükleme formu
- Anket formu tasarımı

## Ödevler
- Detaylı kişisel bilgi formu oluşturun
- Form validasyon kuralları ekleyin
- Farklı input tiplerini içeren interaktif bir form hazırlayın

## Değerlendirme
- Form projesi değerlendirmesi
- Validasyon kontrolü
