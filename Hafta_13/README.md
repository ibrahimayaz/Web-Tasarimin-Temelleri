# HAFTA 13: JavaScript Form Validasyonu ve İleri Konular

## Teorik İçerik
- Form validasyonu teknikleri
- String metodları (trim, length)
- Karşılaştırma operatörleri
- Koşullu ifadeler (if-else)
- isNaN() fonksiyonu
- Pattern matching ve regex temelleri
- Return statement kullanımı
- Event propagation

## Pratik Uygulamalar
- Kapsamlı form validasyonu
- Email validasyonu
- Telefon numarası validasyonu
- Şifre gücü kontrolü
- Gerçek zamanlı validasyon feedback

## Ödevler
- Tam özellikli bir kayıt formu oluşturun (JavaScript validasyonu ile)
- Kullanıcı dostu hata mesajları ekleyin

## Değerlendirme
- Form validasyon projesi incelemesi
- Kullanıcı deneyimi değerlendirmesi
