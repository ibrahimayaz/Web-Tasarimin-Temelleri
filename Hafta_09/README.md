# HAFTA 9: CSS Arka Plan ve Renkler

## Teorik İçerik
- Background özellikleri
- Background-image kullanımı
- Background-repeat ve background-position
- Background-attachment (fixed, scroll)
- Background-size (cover, contain)
- Background shorthand
- Renk kodları (hex, rgb, rgba)
- Gradient arka planlar

## Pratik Uygulamalar
- Arka plan görselleri ekleme
- Parallax efekti oluşturma
- Multiple background kullanımı
- Gradient tasarımları

## Ödevler
- Görsel açıdan zengin bir landing page tasarlayın
- En az 2 farklı arka plan tekniği kullanın

## Değerlendirme
- Landing page projesi sunumu
- Görsel tasarım değerlendirmesi
