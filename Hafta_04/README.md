# HAFTA 4: HTML Tablolar ve Form Temelleri

## Teorik İçerik
- Tablo yapısı (<table>, <tr>, <td>, <th>)
- Tablo özellikleri (border, cellpadding, cellspacing)
- Hücre birleştirme (colspan, rowspan)
- Tablo başlığı (<caption>)
- HTML formlarına giriş
- Form etiketi ve özellikleri
- Input tipleri: text, password, email

## Pratik Uygulamalar
- Basit ve karmaşık tablo örnekleri
- Hücre birleştirme uygulamaları
- Temel form oluşturma
- Farklı input türleri ile çalışma
- Placeholder kullanımı

## Ödevler
- Ders programı tablosu oluşturun
- Kayıt formu tasarlayın (ad, e-posta, şifre alanları ile)

## Değerlendirme
- Tablo projesi incelemesi
- Form yapısı kontrolü
