# HAFTA 1: Web Teknolojilerine Giriş ve HTML Temelleri

## Teorik İçerik
- Internet ve World Wide Web kavramları
- Web sitesi ve web uygulaması arasındaki farklar
- Front-End ve Back-End kavramları
- Statik ve dinamik içerik farkları
- Responsive web tasarım nedir?
- Geliştirme ortamları: Notepad, Notepad++, Sublime Text
- HTML'e giriş ve temel yapısı

## Pratik Uygulamalar
- Text editor kurulumu ve yapılandırması
- İlk HTML sayfası oluşturma
- HTML dosyası kaydetme ve tarayıcıda açma
- Temel HTML yapısı (<html>, <head>, <body>) kullanımı
- Başlık etiketleri (h1-h6) ile çalışma

## Ödevler
- Kendi bilgisayarınıza bir text editor kurun
- Kişisel tanıtım sayfası oluşturun (başlık ve paragraflar içeren)
