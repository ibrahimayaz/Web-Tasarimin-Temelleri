# Web Tasarım Dersi - 14 Haftalık Program Context.md

## Ders Bilgileri
- **Ders Adı:** Web Tasarım ve Geliştirme
- **Seviye:** Başlangıç - Orta Seviye
- **Süre:** 14 Hafta
- **Haftalık Ders Saati:** 4 saat (2 saat teorik, 2 saat pratik)
- **Hedef Kitle:** Web tasarım ve geliştirme öğrenmek isteyen başlangıç seviyesi öğrenciler

## Genel Amaç
Bu ders, öğrencilere modern web tasarımı ve geliştirme konusunda temel bilgi ve becerileri kazandırmayı amaçlamaktadır. Ders sonunda öğrenciler responsive web siteleri tasarlayabilecek, HTML, CSS ve JavaScript kullanarak dinamik web uygulamaları geliştirebilecek düzeye geleceklerdir.

## Öğrenme Çıktıları
Dersi başarıyla tamamlayan öğrenciler:
1. Web teknolojilerinin temel kavramlarını ve çalışma prensiplerini açıklayabilecek
2. HTML5 ile yapılandırılmış, semantik web sayfaları oluşturabilecek
3. CSS3 ve W3.CSS ile modern, responsive tasarımlar geliştirebilecek
4. JavaScript ile dinamik ve interaktif web uygulamaları oluşturabilecek
5. Form validasyonu ve kullanıcı etkileşimi sağlayabilecek
6. AngularJS framework'ü ile temel web uygulamaları geliştirebilecek
7. Modern web standartlarına uygun projeler tasarlayıp yayınlayabilecek
