# HAFTA 10: CSS Box Model ve Pozisyonlama

## Teorik İçerik
- Box model kavramı
- Margin, border, padding özellikleri
- Width ve height
- Box-sizing
- Border özellikleri ve border-radius
- Display özellikleri (block, inline, inline-block, flex)
- Float özelliği
- Position (static, relative, absolute, fixed, sticky)
- Overflow özellikleri

## Pratik Uygulamalar
- Box model manipülasyonu
- Float ile layout oluşturma
- Position ile konumlandırma
- Responsive kutu tasarımları

## Ödevler
- 3 kolonlu bir layout tasarlayın (float kullanarak)
- Sticky navigation menü oluşturun

## Değerlendirme
- Layout projesi kontrolü
- Pozisyonlama becerisi değerlendirmesi
